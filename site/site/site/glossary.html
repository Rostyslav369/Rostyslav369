Glossary

HTML
HyperText Markup Language used to create webpages.
CSS
Cascading Style Sheets used to style HTML content.
JavaScript
A scripting language to create dynamic webpage content.
Web Hosting
A service that allows websites to be accessible on the internet.
Responsive Design
A design approach that adapts to different screen sizes and devices.
