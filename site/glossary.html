Glossary

HTML
Hypertext Markup Language, used to structure web pages.
CSS
Cascading Style Sheets, used for styling web pages.
JavaScript
A programming language that adds interactivity to web pages.
URL
Uniform Resource Locator, the address of a webpage.
DOM
Document Object Model, the structure of an HTML page as a tree.
